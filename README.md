# suttda_game

20장의 화투패에서 각자 총 3장의 카드를 받고 배팅하여 누가 족보가 더 높은지로 승부하는 게임입니다. 각자 처음에 2장의 패를 받고 배팅한 후 1장을 더 받고나서 마지막으로 어떠한 족보를 선택할지 고릅니다.
